üéØ What is Linear Regression?

Linear regression is one of the most fundamental algorithms in machine learning. It finds the best straight line that fits through your data points.

Formula: Y = mX + b

  • Y = predicted output
  • X = input feature
  • m = slope (how steep the line is)
  • b = intercept (where the line crosses the Y-axis)
